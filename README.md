molecbio
========

Molecular biology informatics functions

INSTALLATION
============

You can install this package using pip with the command:
pip install git+https://github.com/dave-the-scientist/molecbio.git

If you have an earlier version installed, you may need to first uninstall it with:
pip uninstall -y molecbio

If you have a very early version, you may need to uninstall it by manually deleting the molecbio folder. You can find where it is installed by importing it in a Python session, and examining the variable `molecbio.__file__`.
